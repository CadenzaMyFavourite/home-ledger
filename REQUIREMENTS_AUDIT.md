# HomeLedger 需求差距审计

审计日期：2026-07-10

本文件按原始产品规格核对当前代码、测试和 Windows 发布证据。`已完成` 表示存在可运行实现与相应验证；`部分完成` 不以数据库表或占位界面冒充完整用户流程。

## 已完成的核心闭环

- 整数最小货币单位的收入、支出、转账 CRUD；状态与转账统计不变量有数据库测试。
- 分类、子分类、支付方式、成员、地点、模板、常用建议、保存筛选、批量删除与批量编辑；批量补丁只更新勾选字段，冲突逐笔返回，成功子集按 correlation ID 审计并可安全撤销。
- 月/周/日日历、事件 CRUD、事件—交易关联、周期计划交易、提醒和桌面通知入口。
- Dashboard、月度/年度确定性统计、分类/支付方式/成员汇总、审核候选、CSV/Excel/打印导出。
- 本地 Ollama/OpenAI-compatible 配置、聚合总结、逐笔建议审核、税务候选提示与 Canada/Ontario 资料包。
- CSV 预览、映射、整数金额解析、重复候选确认、提交与整批撤销。
- 版本化完整备份、SHA-256、SQLite 完整性预检、恢复前备份、重启原子切换与失败回滚。
- 后台计划备份、间隔与保留策略；自动清理不会触及手动或恢复前备份。
- 交易与事件附件管理：原生文件选择、本地托管复制、大小/扩展名/哈希校验、逻辑 ID 访问、解除关联和审计；备份只收录数据库中的有效托管附件。
- 安全自然语言查询：本地模型只接收问题与选项清单，输出严格过滤 DSL；Zod 与 Rust 双重验证字段、枚举、范围、数量和有效 ID，用户确认后才复用参数化交易查询。
- 分组全局搜索：本地参数化查询覆盖账目、事件、普通/税务标签和附件名，通配符按字面量转义；结果分页并可深链到账目、事件或所属附件详情。
- 年度日历概览：12 个月只读网格、重要事件/账单/税务与待处理提醒的非颜色标记、年度切换、键盘操作和月份跳转。
- 每日备注与附件：日期详情提供创建、读取、版本化编辑和软删除；保存后可使用原生选择器关联托管附件，删除存在附件的备注会安全拒绝，数据库和附件均进入完整备份。
- Tauri/WebDriverIO 桌面 E2E：Windows debug Tauri 二阶段自动化覆盖真实 SQLite CSV 导入、真实通知权限/投递命令、备份暂存、应用重启边界恢复和恢复后数据库校验；测试使用隔离 app id 与临时 WDIO capability，日志和截图保存在 `artifacts/desktop-e2e`。
- Windows x64 MSI/NSIS 构建；NSIS 安装、冷启动、覆盖升级、卸载及用户数据库保留演练。
- 前端单元测试、Rust 单元/集成测试、9 条 Playwright 浏览器预览主流程、4 条 WebDriverIO/Tauri 桌面主流程和 50,000 行 SQLite 基准。
